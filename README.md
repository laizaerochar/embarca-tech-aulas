# EmbarcaTech 2026

Repositório criado para armazenar minhas anotações, resumos e aprendizados ao longo do curso **Capacitação Profissional em Sistemas Embarcados (EmbarcaTech)**.

## Sobre o repositório

Aqui você vai encontrar:

- Anotações das aulas organizadas por unidades
- Resumos dos principais conceitos
- Insights e explicações simplificadas

A ideia é transformar o conteúdo do curso em algo mais didático e fácil de revisar para o projeto final.

## Objetivos

- Consolidar meu aprendizado em sistemas embarcados  
- Criar um material de revisão para o futuro  
- Compartilhar conhecimento com as pessoas
- Desenvolver o hábito de documentação técnica
